# CUNY-Expenditure-Analysis-Dashboard
Excel dashboard analyzing CUNY community college expenditure trends by fiscal year and funding source.

## Project Overview
This project analyzes CUNY community college expenditure trends by fiscal year and funding source. The goal is to create a simple dashboard that helps understand how expenditures changed over time and which funding sources contributed the most.

## Tools Used
- Microsoft Excel
- PivotTables
- PivotCharts
- Data Cleaning
- Financial Analysis
- Dashboard Reporting

## Dataset
The dataset used is CUNY Community College Expenditures By Source from NYC Open Data.

## Dashboard Features
- Total expenditure by fiscal year
- Expenditure by funding source
- Funding source trend over time
- Cleaned ACTUAL expenditure data for analysis

## Key Insights
- Total expenditures increased over time.
- City Tax Levy and Tuition were major funding sources.
- The dashboard provides a simple decision-support view of CUNY expenditure trends.
